# 📊 Customer Behaviour Analysis using Python

## 📌 Project Overview

Customer behavior analysis is a crucial aspect of modern business intelligence. It helps organizations understand purchasing patterns, customer preferences, and factors influencing customer retention. This project performs an in-depth analysis of customer transaction data to uncover actionable insights into purchasing behavior, revenue generation, customer segmentation, and churn.

Using **Python** and **Exploratory Data Analysis (EDA)** techniques, this project transforms raw customer data into meaningful business insights that support data-driven decision-making.

---

## 🎯 Project Objectives

- Analyze customer demographics and purchasing behavior.
- Identify the most profitable product categories.
- Study customer churn patterns.
- Analyze payment method preferences.
- Discover monthly revenue trends.
- Segment customers using **RFM Analysis**.
- Generate business recommendations based on data insights.

---

## 📂 Dataset Information

The dataset includes customer purchasing records containing information such as:

- Customer ID
- Gender
- Age
- Product Category
- Purchase Amount
- Payment Method
- Purchase Date
- Churn Status

Additional analytical features were created during preprocessing to support deeper business analysis.
